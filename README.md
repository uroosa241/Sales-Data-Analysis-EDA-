# Sales Data Analysis (EDA)

##  Project Overview

This project performs Exploratory Data Analysis (EDA) on a Diwali Sales dataset using Python. The objective is to clean the data, handle missing values, detect and remove outliers, and generate meaningful business insights through visualizations.

##  Objectives

* Understand customer purchasing behavior.
* Clean and preprocess raw sales data.
* Handle missing values and duplicate records.
* Detect and remove outliers.
* Analyze sales trends across different demographic groups.
* Visualize patterns to support business decision-making.

## 🔄 Data Cleaning Process

### 1. Data Inspection

* Checked column names and data types.
* Examined dataset structure and summary statistics.

### 2. Missing Value Treatment

* Identified missing values.
* Removed rows with missing values in the `Amount` column due to a very low missing percentage.

### 3. Column Removal

Removed unnecessary columns:

* Status
* unnamed1

### 4. Duplicate Handling

* Identified duplicate records.
* Removed duplicate rows.

### 5. Standardization

* Converted column names to lowercase.
* Replaced spaces with hyphens for consistency.

### 6. Data Type Conversion

Converted important columns into appropriate numeric data types.

### 7. Outlier Detection & Removal

Used the **Interquartile Range (IQR)** method to identify and remove outliers from the `Amount` column.

##  Exploratory Data Analysis

The following analyses were performed:

### Sales Distribution

* Histogram with KDE plot for sales amount distribution.

### Gender Analysis

* Total sales by gender.

### Age Group Analysis

* Sales comparison across age groups.
* Spending behavior by age bins.

### Marital Status Analysis

* Relationship between marital status and order volume.

### State Analysis

* Sales amount comparison across states.

### Product Category Analysis

* Total sales generated by each product category.

### Zone Analysis

* Order distribution across zones.

### Correlation Analysis

* Correlation matrix and heatmap for numerical features.

##  Key Insights

* Customer spending patterns vary across age groups.
* Certain product categories generate significantly higher revenue.
* Sales performance differs across regions and zones.
* Gender and marital status influence purchasing behavior.
* Correlation analysis helps identify relationships among numerical variables.
